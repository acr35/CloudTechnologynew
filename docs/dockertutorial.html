Docker Tutorial

What Is Meant By Installing Docker?

When one installs Docker, they are essentially installing the Docker Desktop. Docker Desktop comprises of the following elements:

  • Docker Engine
  • Docker CLI Client
  • Docker Compose
  • Kubernetes
  • Credential Helper
System Requirements:

Before you start to install Docker Desktop, you must make sure you have the following pre-requisites on your computer:

  • Windows 10 64-bit: Pro, Enterprise, or Education (build 16299 or later)
  • Hyper-V and Containers Windows features must be enabled
  • 64-bit processor with Second Level Address Translation (SLAT)
  • 4GB System RAM
  • BIOS-level hardware virtualization support must be enabled in the BIOS settings.
Creating A Docker Repository & Storing an Image:
  • Step 1: Click on the Windows Icon and type "CMD" into the search bar
  • Step 2: Right-click on "Command Prompt" and select the "Run as administrator" option
  • Step 3: After the Command Prompt instance opens up, type "docker pull ubuntu" and press the ENTER key
  • Step 4: After Docker is done pulling the ubuntu image, type "docker run -it ubuntu:latest /bin/bash". This command makes a container to help you run the ubuntu image.
  • Step 5: After the Docker container for the ubuntu image is made, type the command "apt-get update"
  • Step 6: After the Ubuntu updates are installed, install the Apache webserver using the command "apt-get install apache2". Fill out additional information that you will be asked (timezone, geolocation)
  • Step 7: Once the Apache2 webserver has been installed, download a document editor using the command "apt-get install vim"
  • Step 8: Now that vim is installed, open another Command Prompt window (run it as administrator). Type in "docker ps"
  • Step 9: Since this our first time using docker you should only see 1 container, but in the instance this isn’t your first time with docker look for the docker container with “ubuntu:latest” in it’s name and copy the container ID
  • Step 10: To make a new Apache webserver Image, type "docker commit <Container ID> <new image name>"
  • Step 11: You can now close the original Command Prompt window that was used
  • Step 12: Now it is time to boot up our Apache Web Server. We will do so with this command “docker run -it -p 127.0.0.1:80:80 -v <FolderOnMachine>:/var/www/html new image name /bin/bash”
  • Step 13: After this is done, an Apache2 instance will boot up; type "cd /var/www/html"
  • Step 14: To make a file, type "vim.<filename>.html"
  • Step 15: Type in basic html site content (html tag, head tag, body tag, words in a paragraph tag, etc)
  • Step 16: When you are done adding html site content, press the ESCAPE/ESC key, then type ";wq", and then press the ENTER key
  • Step 17: Type the command "cd /"
  • Step 18: Type “etc/init.d/apache2 restart”
  • Step 19/Last Step: View your website by going into your web browser of choice, typing "127.0.0.1", and pressing the ENTER key
